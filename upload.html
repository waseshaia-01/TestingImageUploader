Image Upload Troubleshooter

Identify and fix common image upload issues

Common Reasons for Upload Failures

1. Incorrect Supabase Configuration

Your Supabase project might not be properly configured for storage access.

  • Check if your Supabase URL and API key are correct
  • Verify that the "images" bucket exists in your Supabase storage
  • Ensure your bucket has the right permissions (set to public if needed)

2. CORS Issues

Cross-Origin Resource Sharing (CORS) policies might be blocking your requests.

  • Check your Supabase storage CORS settings
  • Ensure your domain is allowed in the CORS configuration

3. File Size Limitations

Supabase has default file size limits for uploads.

  • Check if your image exceeds the size limit (default is 50MB for free tier)
  • Resize large images before uploading

4. Authentication Issues

Your API key might have insufficient permissions.

  • Verify that your anon key has the right storage permissions
  • Check if your Supabase project has Row Level Security (RLS) enabled
