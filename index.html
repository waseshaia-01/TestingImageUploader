Face Scanner

Advanced facial recognition attendance system

How to use:

  1. Make sure you're in a well-lit environment
  2. Allow camera access when prompted
  3. Position your face in the frame
  4. The system will automatically detect and verify your identity

Camera Access Required

This application needs camera access to function properly. Please allow camera permissions when prompted.

If you've previously denied access:

  • Look for the camera icon in your browser's address bar
  • Click it and select "Always allow"
  • Refresh the page and try again
